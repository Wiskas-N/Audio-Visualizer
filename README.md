# Audio Visualizer

Это аудио-визуализатор на базе Web Audio API и Canvas.
Он захватывает звук с микрофона, разбивает его на частоты и отображает их в виде анимированной графики.

## Описание

Приложение анализирует входящий аудиосигнал в реальном времени и визуализирует спектр частот с помощью символьной матрицы.
Интенсивность и положение символов зависят от амплитуды частот.

В коде добавлены подробные комментарии:

* объясняют логику работы
* указывают, где менять параметры
* показывают, как настраивать визуализацию под свои задачи

Важно: комментарии написаны на скорую руку и могут содержать неточности или ошибки.

## Запуск

1. Склонировать репозиторий
2. Открыть `index.html` в браузере
3. Разрешить доступ к микрофону

## Настройка

Основные параметры, которые можно менять в коде:

* `chars` — набор отображаемых символов
* `fontSize` — размер текста и плотность колонок
* `fftSize` — детализация спектра
* `smoothingTimeConstant` — сглаживание анимации
* цвет и прозрачность в `ctx.fillStyle`
* фон (`background.png`)

Все ключевые места снабжены комментариями.

## Ограничения

* требуется доступ к микрофону
* автозапуск может блокироваться браузером
* производительность зависит от устройства
