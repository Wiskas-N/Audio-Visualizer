# Audio-Visualizer
Это аудио-визуализатор. Он берёт звук с микрофона, разбивает его на частоты и отображает их на экране.  В коде добавлены подробные комментарии с объяснением логики работы и указаниями, где и как изменять параметры и поведение визуализации.  Кто будет использовать: разработчики и дизайнеры как визуальный эффект для сайтов и приложений.
