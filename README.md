# Audio Visualizer

Аудио-визуализатор на базе Web Audio API и Canvas.
Захватывает звук с микрофона, разбивает его на частоты и отображает в виде анимированной графики.

## Структура проекта

```
/project
│
├── index.html   # разметка и подключение файлов
├── styles.css   # стили страницы и canvas
├── script.js    # логика визуализации и работа с аудио
└── background.png (опционально)
```

## Описание

Приложение анализирует входящий аудиосигнал в реальном времени и визуализирует спектр частот с помощью символьной матрицы.

В коде добавлены подробные комментарии:

* объясняют, как работает каждая часть
* показывают, где менять параметры
* помогают быстро адаптировать визуализацию под свои задачи

Важно: комментарии написаны на скорую руку и могут содержать неточности или ошибки.

1. Склонировать репозиторий
2. Убедиться, что все файлы находятся в одной директории
3. Открыть `index.html` в браузере
4. Разрешить доступ к микрофону

## Настройка

Основные параметры находятся в `script.js`:

* `chars`
  Набор символов для отображения

* `fontSize`
  Размер шрифта и плотность колонок

* `analyser.fftSize`
  Детализация спектра (128–512+)

* `analyser.smoothingTimeConstant`
  Сглаживание анимации

* `ctx.fillStyle`
  Цвет символов

* `background.png`
  Фоновое изображение (можно удалить или заменить)

Большинство параметров снабжены комментариями прямо в коде.
